# Synthesis Criteria: How to Identify a Good Game Idea

Use this reference during Step 4 (synthesis) to decide which ideas are worth proposing.

## Source Reliability Hierarchy

| Source | What it proves | Reliability |
|--------|---------------|-------------|
| Reddit pain-point posts (high engagement) | Real, widespread player need | High |
| Reddit requests with many upvoted comments | Shared desire, not just one person | High |
| HN Show HN with 100+ points | Concept resonates with technical early adopters | High |
| HN business discussion with revenue numbers | Monetization model is proven | High |
| Indie Hackers success story in adjacent niche | Revenue path exists | High |
| Single low-engagement Reddit post | Possible niche, unvalidated | Low |
| HN comments expressing interest | Qualitative signal only | Medium |

## Cross-Source Validation

An idea gains confidence when multiple independent sources point to the same gap:

- **Reddit + Indie Hackers**: players want it AND someone has made money in the space → strongest signal
- **Reddit + HN**: players want it AND a dev shipped something similar → technically feasible
- **HN + Indie Hackers**: something shipped AND revenue was reported → proven model, check if market is saturated
- **Single source only**: surface as a weaker idea; note the limited validation

## What Makes a Gap Worth Proposing

A game gap is worth proposing if it satisfies at least two of these:

1. **Stated demand**: players explicitly asked for this type of game ("I wish there was...")
2. **Monetization frustration**: players are unhappy with how existing games monetize (ads, gacha, energy)
3. **Niche but active community**: a subreddit or HN thread exists and is engaged, but the genre is underserved
4. **Adjacent success**: a similar (not identical) game earned measurable revenue on Indie Hackers or HN

## Red Flags: Ideas to Deprioritize

- The top Reddit post in the genre is satisfied ("finally a game that does X" about an existing game)
- HN or Indie Hackers shows multiple well-funded competitors launched recently
- The niche requires live ops, server infrastructure, or large art budgets
- The gap is very small (< 500 engagement across all sources)

## How to Align with `{game_overview}`

After identifying validated gaps, filter for alignment with the user's concept:

- Same genre or mechanic family? → high alignment
- Same target platform (mobile / Steam)? → required for platform-specific proposals
- Matches user's stated theme or art direction? → prefer, but not required
- Contradicts user's explicit constraints (e.g. "no PvP")? → exclude

Propose 3–5 ideas ordered from most to least aligned with `{game_overview}`.

## Signal Strength Labels (use in report)

Annotate each idea's evidence section with one of:

- **Strongly validated** — multiple high-engagement sources confirm the gap
- **Moderately validated** — one strong source + one supporting signal
- **Speculative** — interesting gap but limited community evidence
